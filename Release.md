# LocalControl v1.0.0

First stable release: a local UNIX-socket control interface for Limnoria bots
with a predictable CLI, secure hostmask-mapped execution, and zero remote
administrative exposure.

**What's included:**

- LocalControl plugin for dedicated UNIX socket handling
- `botctl` CLI client with `bot` and `exec` command modes
- Synthetic identity mapped through Limnoria hostmasks for secure local control
- Full documentation with setup, usage, troubleshooting, and multi-bot wrappers
- Stable API ready for real-world use
